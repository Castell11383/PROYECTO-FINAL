Descripción del producto

Este esfigmomanómetro completamente automático dispone de todas las funciones necesarias para una confortable, rápida y precisa medición de la presión sanguínea.


Este modelo dispone del sistema Omron Intelli-sense. Gracias a este sistema la presión del inflado nunca será demasiado alta o demasiado baja. En caso de un latido irregular o un movimiento del paciente mientras se esté realizando la medición, se mostrará un aviso en pantalla.



  • Memoria para 21 mediciones.

  • Indicador de hipertensión.
